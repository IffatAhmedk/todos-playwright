# todos-playwright

## Prerequisites:
  - Node.js v18+
  - Playwright installed (npm i -D @playwright/test)
  - Tests in tests/ or root directory

## Install dependencies: 
`npm install`

## Run Tests:
`npx playwright test`

## Run with Traceviewer: 
`npx playwright test --ui`

## View Report:
`npx playwright show-report`
